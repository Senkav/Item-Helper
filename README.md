# Item Helper — Dota 2 Lua Script

Скрипт читает вражеский пик и рекомендует предметы прямо во время игры. Панель с советами отображается на экране в реальном времени.

---

## Возможности

### Анализ вражеской команды

Каждые 2 секунды скрипт считывает героев противника и строит карту угроз: инвизы, контроль, бёрст, хил, иллюзии и прочее. На основе этого каждый предмет получает оценку.

### Рекомендации предметов

Список из 3–10 предметов, отсортированных по приоритету. Для каждого показывается:

- категория — `MUST` / `SITUATIONAL` / `LUXURY`
- причина рекомендации
- стоимость и статус «куплен / хватает золота»
- мини-чипы с разбором оценки

### Учёт роли и стиля героя

Скрипт знает роль вашего героя (carry, mid, offlane, support, hardsupport) и его стиль (phys / magic / utility / hybrid). Предметы, которые не подходят для вашей роли, получают штраф и не попадают в топ.

### Специфика конкретных героев

Отдельная база для героев с уникальной механикой: Arc Warden, Phantom Lancer, Meepo, Terrorblade и других. Например, Battle Fury на Arc Warden получает штраф, потому что Tempest Double его не копирует.

### Контрпик-предметы

Отдельная секция с предметами, которые прямо контрят конкретных вражеских героев. Для каждого предмета показано, скольких героев он закрывает.

### Нейтральные предметы

Рекомендации по нейтральным предметам текущего тира (T1–T5 в зависимости от времени игры).

### Анализ нетворса

Полоска сравнения нетворса команд с индикатором темпа (`AHEAD` / `EVEN` / `BEHIND`). Влияет на веса предметов: если вы впереди, предлагаются дорогие carry-предметы; если позади, акцент на дешёвые и защитные.

### Enemy Focus

Строки с вражескими героями и лучшим предметом против каждого из них.

### Трекинг предметов врагов

Скрипт видит, что уже купили враги (BKB, Linken, Aeon), и корректирует рекомендации. Например, если у нескольких врагов есть BKB, Abyssal Blade и Nullifier получают бонус.

### Режимы игры

Поддерживаются: Ranked, Turbo, All Pick, Captain's Mode, Single Draft, All Random, Ability Draft. В Turbo акцент смещается с ранних предметов на поздние.

### Локализация

Три языка: English, Русский, 中文. Язык подхватывается автоматически из настроек чит-меню.

---

## Настройки

Всё настраивается через меню `General → Main → Item Helper`:

- включение / выключение, горячая клавиша для скрытия панели
- режим отображения: всегда / только в чит-меню / только в магазине
- расширенный или минимальный режим панели
- масштаб (60–200%), прозрачность
- фильтр врагов: анализировать всех / только выбранных / исключить выбранных
- автоанализ, максимум предметов, показ причин, угроз, бейджей, breakdown-чипов
- количество строк Enemy Focus

Панель перетаскивается мышью за шапку. Позиция сохраняется между сессиями.

---

## Требования

Движок с Lua-скриптингом, поддерживающий стандартные API: `Heroes`, `NPC`, `Entity`, `Render`, `Menu`, `Input`, `Engine`, `Panorama`.
