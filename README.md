# Item-Helper
Item Helper — Dota 2 Lua Script
Скрипт автоматически анализирует вражеский пик и рекомендует предметы в реальном времени. Работает внутри игры, отображает панель с советами прямо на экране.
Возможности
Анализ вражеской команды
Скрипт читает героев противника каждые 2 секунды и строит карту угроз — определяет сколько у врагов инвизов, контроля, бёрста, хила, иллюзий и прочего. На основе этого выставляет оценку каждому предмету.
Рекомендации предметов
Список из 3–10 предметов, отсортированных по приоритету. Каждый предмет показывает:

категорию (MUST / SITUATIONAL / LUXURY)
причину рекомендации
стоимость и статус "куплен / хватает золота"
мини-чипы с разбором оценки

Учёт роли и стиля героя
Скрипт знает роль вашего героя (carry, mid, offlane, support, hardsupport) и его стиль (phys / magic / utility / hybrid). Предметы, плохие для вашей роли, получают серьёзный штраф и не попадают в топ.
Специфика конкретных героев
Отдельная база для героев с уникальной механикой — Arc Warden, Phantom Lancer, Meepo, Terrorblade и других. Например, BF на Arc Warden получает штраф, потому что Tempest Double его не копирует.
Контрпик предметы
Отдельная секция с предметами, которые прямо контрят конкретных вражеских героев. Отображается сколько героев закрывает каждый предмет.
Нейтральные предметы
Рекомендации нейтральных предметов по текущему доступному тиру (T1–T5 в зависимости от времени игры).
Анализ нетворса
Полоска сравнения нетворса команд с индикатором темпа игры (AHEAD / EVEN / BEHIND). Влияет на веса предметов — если вы за, предлагаются дорогие carry-предметы; если против, акцент на дешёвые защитные.
Enemy Focus
Строки с вражескими героями и лучшим предметом против каждого из них.
Трекинг предметов врагов
Скрипт видит что уже купили враги (BKB, Linken, Aeon) и корректирует рекомендации — например, при нескольких BKB у врагов Abyssal Blade и Nullifier получают бонус.
Определение режима игры
Поддерживает Ranked, Turbo, All Pick, Captain's Mode, Single Draft, All Random, Ability Draft. Turbo сдвигает акцент с ранних предметов на поздние.
Локализация
Полная поддержка трёх языков: English, Русский, 中文. Язык подхватывается из настроек чит-меню автоматически.
Настройки
Всё настраивается через меню (General → Main → Item Helper):

включение/выключение, горячая клавиша для скрытия панели
режим отображения: всегда / только в чит-меню / только в магазине
расширенный или минимальный режим панели
масштаб (60–200%), прозрачность
фильтр врагов — анализировать всех / только выбранных / исключить выбранных
автоанализ, максимум предметов, показ причин, угроз, бейджей, breakdown-чипов
количество строк Enemy Focus

Панель перетаскивается мышью за шапку, позиция сохраняется между сессиями.
Структура кода
item_helper.lua
├── Локализация (L_STRINGS, L_REASONS, L_REASON_SLANG, L_NEUTRAL_REASONS)
├── HERO_TAGS           — теги угроз по героям (~130 героев)
├── HERO_ROLES          — роль и стиль каждого героя
├── ITEM_ROLE_PENALTY   — штрафы предметов по роли/стилю
├── HERO_SPECIFIC_ITEMS — бонусы/штрафы предметов для конкретных героев
├── HERO_COUNTERS       — прямые контрпики по героям
├── ITEM_COSTS          — стоимости предметов для нетворс-расчётов
├── ITEM_DB             — основная база предметов (~80 предметов)
├── NEUTRAL_DB          — нейтральные предметы T1–T5 (~60 предметов)
├── COUNTER_RULES       — правила контр-логики (тег угрозы → тег предмета)
├── Движок анализа      — computeItemScore, analyzeEnemyTeam и др.
└── Рендер панели       — drawPanel, drawHeader, drawItemCard и др.
Требования
Движок с Lua-скриптингом, поддерживающий стандартные API: Heroes, NPC, Entity, Render, Menu, Input, Engine, Panorama.
